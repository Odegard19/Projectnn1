ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРОГРАММЫ "SMART TASK MANAGER"
===================================================

Для запуска программы вам не нужны специальные знания программирования. Просто следуйте этим простым шагам:

ШАГ 1: ПОДГОТОВКА (Требуется только Python)
Убедитесь, что на компьютере установлен Python (желательно версии 3.8 и новее). 
*Если вы устанавливаете Python впервые, обязательно поставьте галочку "Add Python to PATH" в самом первом окне установщика!

ШАГ 2: БЫСТРЫЙ ЗАПУСК
1. Откройте эту папку (SmartTaskManager).
2. Найдите файл с названием "start.bat" (или просто "start", если у вас скрыты расширения).
3. Дважды кликните по нему левой кнопкой мыши.

Что произойдет дальше:
На секунду откроется черное окно консоли. Скрипт сам проверит, установлена ли библиотека для красивого дизайна (CustomTkinter), и если ее нет — автоматически скачает ее из интернета. Сразу после этого откроется главное окно программы.

ШАГ 3: ИСПОЛЬЗОВАНИЕ
Вы можете смело добавлять задачи. При первом добавлении в этой папке автоматически появится файл "tasks.json". В нем хранятся все ваши записи, поэтому после закрытия программы ничего не потеряется.


---------------------------------------------------
АЛЬТЕРНАТИВНЫЙ ЗАПУСК (Если start.bat не сработал)
Если по какой-то причине двойной клик не открывает программу:
1. Кликните правой кнопкой мыши по пустому месту в этой папке и выберите "Открыть в терминале" (или "Open in Terminal").
2. Введите команду установки дизайна: 
   python -m pip install customtkinter
   (и нажмите Enter)
3. Запустите саму программу командой:
   python app.py
   (и нажмите Enter)
